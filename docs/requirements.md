# PDF 分割工具 - 需求文档

## 1. 项目概述

一个基于 Web 的 PDF 分割工具，用户通过浏览器上传 PDF 文件，系统自动读取页数并支持按页码区间进行分割，分割前可预览页面内容，分割后可自定义重命名输出文件。

## 2. 用户角色

| 角色 | 描述 |
|------|------|
| 普通用户 | 上传 PDF、选择分割区间、预览、下载分割后的文件 |

## 3. 功能需求

### 3.1 PDF 上传

- **FR-01**: 支持拖拽上传和点击上传两种方式
- **FR-02**: 仅接受 `.pdf` 格式文件，前端校验 + 后端校验
- **FR-03**: 上传成功后显示文件名和文件大小
- **FR-04**: 文件大小限制 600MB，超出时给出提示

### 3.2 页数读取

- **FR-05**: 上传成功后自动读取 PDF 总页数
- **FR-06**: 页数信息在界面上清晰展示（如"共 50 页"）
- **FR-07**: 读取失败时显示错误信息，提示用户检查文件是否损坏

### 3.3 页码区间选择

- **FR-08**: 提供两种分割模式：
  - **等分模式**：输入 N 份，自动将 PDF 按页数均分为 N 个区间
  - **自定义模式**：手动输入起止页码，支持添加多个区间
- **FR-09**: 等分模式自动计算区间，剩余页面合并到最后一个区间
- **FR-10**: 自定义模式下，用户可以添加/删除区间行
- **FR-11**: 每个区间格式为 `[起始页, 结束页]`，起始页 ≤ 结束页
- **FR-12**: 页码不能超过总页数，不能重叠，前端实时校验
- **FR-13**: 区间列表支持拖拽排序（可选）

### 3.4 预览功能

- **FR-14**: 每个分割区间旁有"预览"按钮
- **FR-15**: 点击预览后展示该区间首页的缩略图
- **FR-16**: 预览支持翻页查看区间内的所有页面（以缩略图或画布形式）
- **FR-17**: 预览弹窗/侧栏，不阻塞主操作流程

### 3.5 分割与重命名

- **FR-18**: 每个区间有一个可编辑的输出文件名输入框
- **FR-19**: 默认文件名格式：`{原文件名}_part{N}_{start}-{end}.pdf`
- **FR-20**: 用户可自定义每个输出文件名，支持批量命名模板（如 `{original}_section_{index}`）
- **FR-21**: 点击"执行分割"按钮，后端按区间切割 PDF
- **FR-22**: 分割完成后返回结果列表，包含每个文件的状态（成功/失败）

### 3.6 下载

- **FR-23**: 支持单个文件下载
- **FR-24**: 支持全部文件打包为 ZIP 下载
- **FR-25**: 下载文件名使用用户自定义的名称

### 3.7 重置

- **FR-26**: 提供"重置"按钮，清除当前文件及所有操作状态

## 4. 非功能需求

### 4.1 性能

- **NFR-01**: PDF 上传 600MB 以内文件在 3 分钟内完成
- **NFR-02**: 页数读取在 3 秒内完成
- **NFR-03**: 分割操作在 60 秒内完成（100 页以内 PDF）
- **NFR-04**: 预览图加载在 2 秒内完成

### 4.2 可用性

- **NFR-05**: 界面简洁直观，主要操作流程 ≤ 3 步
- **NFR-06**: 表单验证实时反馈，错误信息明确
- **NFR-07**: 支持主流浏览器（Chrome、Edge、Firefox 最新版）

### 4.3 安全性

- **NFR-08**: 后端校验文件类型（MIME + 扩展名）
- **NFR-09**: 上传文件存储在临时目录，处理后定期清理
- **NFR-10**: 文件名过滤特殊字符，防止路径穿越

### 4.4 兼容性

- **NFR-11**: 后端支持 Python 3.9+
- **NFR-12**: 前端无框架依赖，纯 HTML/CSS/JS 实现

## 6. 优先级

| 优先级 | 功能 |
|--------|------|
| P0（MVP） | PDF上传、页数读取、自定义区间分割、重命名、单个下载 |
| P1 | 等分模式、批量命名、ZIP打包下载、预览 |
| P2 | 拖拽上传、拖拽排序区间、批量预览 |
